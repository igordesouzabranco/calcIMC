# calcIMC

A simple and efficient BMI (Body Mass Index) calculator application.

## Overview

calcIMC is a lightweight tool designed to help users quickly calculate their Body Mass Index (BMI) based on their height and weight. This application provides instant results with a user-friendly interface.

## Features

- 📊 Quick BMI calculation
- 🎯 Instant results with health category classification
- 🎨 Clean and intuitive user interface
- 📱 Responsive design
- 🌐 Web-based accessibility

## Getting Started

### Prerequisites

- A modern web browser
- Basic knowledge of HTML, CSS, and JavaScript (if modifying the code)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/igordesouzabranco/calcIMC.git
```

2. Navigate to the project directory:
```bash
cd calcIMC
```

3. Open `index.html` in your web browser or access the live version online.

## How to Use

1. Enter your weight (in kilograms)
2. Enter your height (in centimeters or meters)
3. Click the "Calculate" button
4. View your BMI result and corresponding health category

## BMI Categories

- **Underweight**: BMI < 18.5
- **Normal weight**: BMI 18.5 - 24.9
- **Overweight**: BMI 25 - 29.9
- **Obese**: BMI ≥ 30
